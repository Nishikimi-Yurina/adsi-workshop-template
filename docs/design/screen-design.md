# 画面設計

## 画面一覧

| # | 画面名 | パス | ロール | 説明 |
|---|--------|------|--------|------|
| 1 | ログイン | /login | 全員 | ID/パスワード入力 |
| 2 | ダッシュボード | / | 全員 | 打刻ボタン・今日の勤怠状況・直近の申請状況 |
| 3 | 勤怠一覧 | /attendance | 全員 | 当月の勤怠一覧・打刻修正への導線 |
| 4 | 打刻修正 | /attendance/edit/{id} | 全員 | 自分の打刻時刻を修正 |
| 5 | 休暇申請 | /leave-requests | 全員 | 申請一覧・新規申請フォーム |
| 6 | 残業申請 | /overtime-requests | 全員 | 申請一覧・新規申請フォーム |
| 7 | 承認管理 | /approvals | 管理者 | 未承認の休暇・残業申請を一覧表示・承認/却下 |
| 8 | 月次レポート | /reports/monthly | 全員 | 自分（管理者は全社員）の月次勤務サマリー |

## 画面遷移

```
ログイン → ダッシュボード（打刻もここで行う）
              ├── 勤怠一覧 → 打刻修正
              ├── 休暇申請
              ├── 残業申請
              ├── 承認管理（管理者のみ）
              └── 月次レポート
```

## 主要画面ワイヤーフレーム

### ダッシュボード（打刻機能付き）

```
┌─────────────────────────────────────┐
│ 現在時刻: 09:00:00                    │
│                                     │
│  [出勤する]    [退勤する]              │
│                                     │
│ 今日の勤怠                            │
│ ┌───────────────────────────────┐   │
│ │ 出勤: 09:00  退勤: --:--     │   │
│ │ 勤務時間: 計算中...            │   │
│ └───────────────────────────────┘   │
│                                     │
│ 直近の申請                            │
│ ┌───────────────────────────────┐   │
│ │ 07/10 有給休暇  申請中          │   │
│ │ 07/08 残業申請  承認済み         │   │
│ └───────────────────────────────┘   │
└─────────────────────────────────────┘
```

- 出勤前: [出勤する] が活性、[退勤する] が非活性
- 出勤後: [出勤する] が非活性、[退勤する] が活性
- 退勤後: 両方非活性、勤務時間を表示

```
│ │ 7/1 │ 09:00 │ 18:00 │ 8:00 │ 0  │ [修正]│ │

│ └─────┴───────┴───────┴──────┴────┴────┘ │
└─────────────────────────────────────┘
```
